Utilizar sass: 
0) instalar npm y el node js (esto se hace una sola vez en la pc y punto)

Para agregar al proyecto:
1)En la consola, iniciar en npm (npm init)
2)Enter enter enter enter....
3)Instalar el nodemon (npm install node-sass nodemon) 
4)Agregar las lineas al package.json

"build-css": "node-sass --include-path scss scss/prueba.scss css/prueba.css",
    "watch-css": "nodemon -e scss -x \"npm run build-css\""

(estas lineas van separadas por comas)

entiendo que cada vez que estras al proyecto y vas a editar el sass hay que abrir la consola y desde la carpeta del proyecto ejecutar "npm run watch-css" (sin comillas amigue)
